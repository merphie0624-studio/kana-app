# 靠岸｜彼岸の島（kana-app）

這是一個以「停下來、被接住」為核心的靜態 App。 
不追求效率、不要求完成，只提供一個可以靠岸的地方。

適合在手機上使用。

---

## 核心概念

- **靠岸**：不是答案，也不是終點，只是暫時停一下
- **抽卡**：不是預言，而是一次相遇
- **故事牆**：不是紀錄成果，而是接住當下的自己
- **留白**：什麼都不寫，也是一種回應

---

## 使用流程（實際體驗順序）

1. **第一次進入**
   - 看到迎接文字
   - 點「靠岸」
   - 此畫面只會出現一次（可在「關於」重置）

2. **抽卡頁（今天）**
   - 問句： 
     > 今天的我，想要接住自己什麼？
   - 可選：
     - 抽一張卡（全部系列）
     - 或指定 Flow / Free / For
   - 抽到後顯示：
     - 抽到日期
     - 第幾次相遇
     - 第一次相遇日期

3. **抽卡後三個行動**
   - **回應這張卡** 
     → 進入「我的故事牆」，直接開啟該卡的寫入頁
   - **我的故事牆** 
     → 查看已抽到的卡、回顧自己的紀錄
   - **再次相遇** 
     → 再抽一次（不限制次數）

4. **我的故事牆**
   - 未抽到的卡：
     - 顯示卡背（不偷看正面）
   - 已抽到的卡：
     - 可點開寫下：
       - 文字（可留白）
       - 一張照片（會自動縮小，只存在裝置中）
       - 情緒小符號（單選）
   - 所有內容都只存在於你的裝置（localStorage）

---

## 系列說明（目前）

- **Flow**：流動、開始、春
- **Free**：放開、呼吸、夏
- **For**：收與放、理解、秋
- **Forever**：預留（之後加入）

---

## 技術說明（很簡單）

- 純前端（Static）
- 無後端、無帳號、無追蹤
- 所有資料存在：
  - `localStorage`
- 可匯出 / 匯入 JSON 作為備份
